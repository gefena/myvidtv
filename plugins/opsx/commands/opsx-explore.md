# /opsx-explore

Run the `openspec-explore` workflow for this workspace.

## Arguments

- `request`: idea, problem, comparison, or change name to explore

## Workflow

1. Use the `openspec-explore` skill.
2. Investigate the codebase and relevant OpenSpec artifacts as needed.
3. Do not implement application code in this mode.
4. Offer to move to proposal creation when the problem is clear enough.
